# LiveKit Project

This is a Streamlit application that integrates with LiveKit for real-time communication.

## Setup

1. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Set up environment variables:
   - Copy `.env.example` to `.env`
   - Fill in your actual LiveKit API key and secret in `.env`

3. Run the application:
   ```
   streamlit run app.py
